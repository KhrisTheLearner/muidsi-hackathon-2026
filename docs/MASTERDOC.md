# Project Documentation

## Overview
This document outlines the structure and components of the `muidsi-hackathon-2026` project.

## Directory Structure

### `data/`
This directory contains the data assets used by the project.

*   **`raw/`**: Stores unprocessed, original data files.
    *   `Readme.txt`: Contains metadata, Information Quality Guidelines, and contact details regarding the USDA/NASS Large Datasets. This file serves as the primary documentation for the origin and compliance of the raw data.

## Data Sources

### USDA/NASS
The project utilizes data from the **USDA National Agricultural Statistics Service (NASS)**.
*   **Compliance**: The data adheres to USDA Information Quality Guidelines.
*   **Privacy**: Usage is subject to the Privacy Act of 1974 as detailed in the dataset documentation.
