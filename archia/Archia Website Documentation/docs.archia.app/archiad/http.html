Archia is a runtime environment for agentic workflows. archiad is an HTTP server that provides the functionality of that runtime environment. This allows for organizations to leverage Archia as a service inside their infrastructure as they might use a database or memory cache.

Completions

Generate model responses (OpenAI-compatible)

Create Completion

Creates a completion for the provided prompt and parameters.

Request Body schema: application/json
required
best_of
integer or null <int32> >= 0

Generates best_of completions server-side and returns the "best".

echo
boolean or null

Echo back the prompt in addition to the completion.

frequency_penalty
number or null <float>

Number between -2.0 and 2.0. Positive values penalize new tokens based on their existing frequency in the text so far.

logit_bias
any

Modify the likelihood of specified tokens appearing in the completion.

logprobs
integer or null <int32> >= 0

Include the log probabilities on the logprobs most likely tokens.

max_tokens
integer or null <int32> >= 0

The maximum number of tokens to generate in the completion.

model
required
string

ID of the model to use.

n
integer or null <int32> >= 0

How many completions to generate for each prompt.

presence_penalty
number or null <float>

Number between -2.0 and 2.0. Positive values penalize new tokens based on whether they appear in the text so far.

string or Array of strings or Array of integers or Array of integers (CompletionPrompt)
null or (StopSequence (StopSequence (string) or Array of StopSequence (strings)))
stream
boolean or null

Whether to stream back partial progress.

suffix
string or null

The suffix that comes after a completion of inserted text.

temperature
number or null <float>

What sampling temperature to use, between 0 and 2.

top_p
number or null <float>

An alternative to sampling with temperature, called nucleus sampling.

user
string or null

A unique identifier representing your end-user.
